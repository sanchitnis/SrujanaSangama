# Specification: `kaizen-wellbeing-reva` Plugin

## 1. Goal Description
Detailed specification for the `kaizen-wellbeing-reva` plugin under the SrujanaSangama private marketplace, coordinating self-reflection coaching, work-life balance wheels, daily/weekly GPS planning, and FDP integration.

## 2. Scope Boundaries
- **In scope**: Personal Kaizen coach, GPS planners, habit tracking, and wellbeing indicators (Hitaishin).
- **Out of scope**: Storing raw student/faculty medical data.

## 3. Verification Criteria
1. Manifest files (`plugin.json`, `package.json`) resolve correctly inside the marketplace environments.
