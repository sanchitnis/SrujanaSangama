# Specification: `teaching-learning-reva` Plugin

## 1. Goal Description
Detailed specification for the `teaching-learning-reva` plugin under the SrujanaSangama private marketplace, coordinating syllabus & lesson planning, outcome mapping, active pedagogy strategies, and assessment engineering.

## 2. Scope Boundaries
- **In scope**: Flipped classroom planners, HOTS quiz templates, and Bloom's alignment audits.
- **Out of scope**: Storing student-graded exam results.

## 3. Verification Criteria
1. Manifest files (`plugin.json`, `package.json`) resolve correctly inside the marketplace environments.
