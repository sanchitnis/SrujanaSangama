# Agent Skill Instructions: patent-input

## Purpose
Guide the user to provide a Markdown (.md) file containing their invention writeup. Validate the file and prepare it for extraction.

## Agent Actions
- Prompt the user to upload or specify the path to their Markdown file.
- Validate that the file is in Markdown format and is accessible.
- Confirm successful receipt and readiness for extraction.
- If the file is missing or not in Markdown format, prompt the user to correct it.

## Output
- Path to the validated Markdown file for further processing.
