# Agent Skill Instructions: patent-export

## Purpose
Guide the user to export the draft patent application and prepare for filing.

## Agent Actions
- Instruct the user on how to convert the Markdown draft to DOCX or PDF using available tools (pandoc, VS Code extensions, etc.).
- Provide links to Indian Patent Office resources and official filing instructions.
- Remind the user to review all sections for completeness and compliance before submission.
- Offer tips for successful filing (e.g., formatting, required forms, fees).

## Output
- Exported DOCX/PDF file and guidance for next steps in the Indian patent process.
