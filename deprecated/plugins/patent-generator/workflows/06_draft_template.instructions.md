# Agent Skill Instructions: patent-draft-template

## Purpose
Generate and pre-fill a detailed patent application template in Markdown, using all available extracted and AI-inferred information.

## Agent Actions
- Use all previously extracted and AI-inferred details to pre-fill as much of the template as possible:
  - Title
  - Abstract
  - Problem statement
  - Objectives
  - Technical field
  - Background
  - Key features
  - Advantages
  - Experimental data/results
  - Drawings (descriptions)
  - Claims (drafted from features and inventive aspects)
  - Applicant/inventor details (if available)
- Clearly mark any sections that are missing or need user input.
- Present the draft template to the user for review and completion.

## Output
- Pre-filled Markdown patent application template, with clear prompts for missing information.
