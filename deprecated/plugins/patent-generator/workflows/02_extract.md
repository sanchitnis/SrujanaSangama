# Command: patent-extract

## Purpose
Extract invention details from the provided Markdown file.

## Steps
- Parse the Markdown to extract:
  - Title
  - Abstract
  - Problem statement
  - Objectives
  - Technical field
  - Background
  - Key features
  - Advantages
  - Experimental data/results
  - Drawings (if described)
- Present extracted information to the user for review and correction.
