# /grant-check

Perform a rigorous grant proposal audit using SrujanaShodha's Funding Navigator + Work Product Reviewer agents.

Paste the grant proposal draft below the divider line.

---

## What SrujanaShodha will do

1. **Agency & Scheme Identification**: Identify the target funding agency and scheme. If not stated, ask.
2. **Eligibility Verification**: Check if the PI is eligible for this specific scheme (career stage, domain, institution type).
3. **Proposal Compliance Audit**: Run the Grant Proposal Standard checklist (`rules/GRANT_PROPOSAL_STANDARD.md`).
4. **Funding Navigator Assessment**: Rate thematic alignment, novelty, feasibility, and budget logic.
5. **Work Product Review**: Apply the Strength → Gap → Next Move framework.
6. **SDG Impact Assessment**: Score the SDG claim using `rules/SDG_MAPPING_STANDARD.md`.
7. **Prioritised Revision List**: Numbered list of must-fix items before submission.
