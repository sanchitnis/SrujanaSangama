# /attainment

Verify course or program direct and indirect attainment logs for:
1. direct-sem-attainment: Confirm calculation logic, thresholds, and target levels mapping.
2. indirect-attainment: Ensure indirect surveys (course-end, program-exit) match the required response counts.
3. gaps-and-remediation: Identify program outcomes falling below the target threshold and flag mandatory faculty action items.
