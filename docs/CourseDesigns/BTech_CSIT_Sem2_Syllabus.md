## 1. Internet of Things (IoT)
* **Course Code:** B24EN0101
* **Course Type:** HC (Integrated)
* **Credits:** 2
* **Structure (LTP):** 1-0-1 (Lecture: 1, Tutorial: 0, Practice: 1)
* **Contact Hours:** 14 Hours Lecture, 28 Hours Practical

### Course Overview
The Internet of Things (IoT) expands access to the world-wide web from computers, smartphones, and other typical devices to create a vast network of appliances, toys, apparel, and other goods that are capable of connecting to the Internet. This introductory course focuses on IoT architecture, Boards, sensors, actuators and communication protocols. The course is supported with hands on sessions that incorporates different types of sensors interfaced with loT board to build loT projects to solve real time problems. The case study of deployment of IoT in various applications are provided.

### Course Objectives
1. To understand the fundamentals of Internet of Things.
2. Inculcate knowledge of IoT devices, Sensors and Communication Protocols in various application domains.
3. Gain expertise in interfacing of various sensors to loT Boards, Arduino and Node MCU.
4. Discuss various applications of IoT.

### Course Outcomes (COs)
* **CO1:** Describe loT architecture and Models of IOT.
* **CO2:** Identify and work on different loT development boards.
* **CO3:** Identify communication technologies, protocols, and cloud services.
* **CO4:** Identify sensors & actuators and demonstrate the interfacing of sensors & actuators to loT boards.
* **CO5:** Interpret various Applications of IoT.
* **CO6:** Develop simple loT projects and modules.

### Course Content
* **IoT Basics and Arduino UNO Board:** Introduction to loT and its working, IoT Architecture, Models of IOT, Characteristics of IoT, Merits, demerits, futures scope and challenges of IoT, IOT Echo system, Arduino UNO board, memories, microcontroller, Basic programming of Arduino UNO board: Numeric data types, text data types, Array data types, Digital input output functions. Communication Technologies: Bluetooth, ZigBee, Wi-Fi, Cellular.
* **IoT Enabling Technologies and Applications:** IoT Development Boards: Node MCU, Raspberry Pi; Sensors and Actuators: Temperature Sensor, PIR Sensor, Ultrasonic senor, LDR, Applications of loT, Protocols: HTTP, MQTT, COAP; IoT Cloud Platforms: Arduino Cloud, Thing Speak, Blink Cloud.

### Practice / Lab Sessions
1. **Experiment-01:** Introduction to Arduino Board & getting started with Arduino IDE software. Write a program to blink an LED infinite times or exactly 3 times with configurable delay.
2. **Experiment-02:** Write a program to increase/decrease the brightness of an LED, and control it using a Potentiometer.
3. **Experiment-03:** Interface an LDR to Arduino, display voltage on serial monitor, and control LED brightness based on light intensity.
4. **Experiment-04:** Interface a temperature sensor, display values on the serial monitor, and map ranges onto an LCD screen.
5. **Experiment-05:** Interface an ultrasonic sensor and display the measured distance from an object.
6. **Experiment-06:** Introduction to Node MCU and write a program for a 2-bit counter.
7. **Experiment-07:** Interface a PIR motion sensor to Node MCU to control an LED indicator based on movement detection.
8. **Experiment-08:** Interface a single LED to blink via Node MCU using a Bluetooth module connection.
9. **Experiment-09:** Write a program to control the status of a high-voltage LED/appliance using a Relay.
10. **Experiment-10:** Interface a temperature sensor with Node MCU to log and display values on a Web Page/Cloud platform.

### Sample Project / Case Study Topics
* IoT based Automated Table Lamp / Light Dimmer and Speed Controller
* IoT based Patient Health Monitoring / Heart Attack Detection & Heart Rate Monitor
* IoT based Smart Home Controller Using Blynk / Smart Mirror with News & Temperature
* IoT based Garbage Monitoring / Liquid Level Monitoring System
* IoT based Irrigation Monitoring & Controller System / Air Pollution Meter
* IoT based Electronic Door Opener / Car Parking System

### Challenge Experiments
1. Introduction to ESP module & web server programming using Arduino IDE to control GPIO lines.
2. Formulate HTTP client and server setups to monitor status fields.
3. Implement Publisher/Subscriber data architecture via MQTT protocol.
4. Log live sensor analytics stream into ThingSpeak cloud architecture.

### Text & Reference Books
* **Textbook:** Vijay Madisetti, Arshdeep Bahga, *Internet of Things: A Hands-On-Approach*, Second Edition, 2014.
* **Reference:** Raj Kamal, *Internet of Things: Architecture & design Principle*, McGraw Hill Education, 2017.

## 2. Probability and Statistics
* **Course Code:** B24AS0203
* **Course Type:** HC
* **Credits:** 3
* **Structure (LTP):** 3-0-0
* **Total Classes:** 42

### Course Overview
This course introduces fundamental statistical methods and probability concepts with applications in computing. It covers curve fitting using least squares, correlation and regression techniques, and explores both discrete and continuous probability distributions. Joint probability distributions, stochastic processes, and Markov chains are studied to understand system modeling. The course also introduces sampling theory and hypothesis testing. Special emphasis is given to computer science fields like data science, machine learning, computer vision, computer graphics, and operating systems.

### Course Objectives
1. Learn the concept of curve fitting and structural statistical methods.
2. Understand fundamentals of probability and random variables.
3. Comprehend joint probability structures and stochastic processes.
4. Apply the concept of test of hypothesis across diverse fields of engineering.

### Course Outcomes (COs)
* **CO1:** Construct data models to fit linear and non-linear equations using least squares.
* **CO2:** Compute regression lines for specific statistical data analysis.
* **CO3:** Define probability density functions and random variables for engineering problems.
* **CO4:** Analyze discrete and continuous probability distributions.
* **CO5:** Make use of stochastic processes and Markov chains across transition states.
* **CO6:** Apply the concepts of sampling theory to solve various engineering problems.

### Course Content
* **Curve Fitting:** Curve fitting by the method of least squares and fitting of the curves of the form: $y=ax+b$, $y=ax^{2}+bx+c$, $y=ae^{bx}$, $y=ax^{b}$ and $y=ab^{x}$.
* **Statistical Methods:** Correlation - Karl Pearson's coefficient of correlation; Regression analysis - lines of regression; Rank correlation. (*Data Science Applications*)
* **Probability Distributions:** Random variables, Discrete and continuous probability distributions. Binomial, Poisson, Normal distributions, and Exponential distributions. (*Applications in Machine Learning, Computer Vision, and Computer Graphics*)
* **Joint Probability Distribution & Stochastic Processes:** Joint Probability distribution for two discrete random variables, expectation, covariance, correlation coefficient. Stochastic processes, probability vector, stochastic matrices, fixed points, regular stochastic matrices, Markov chains, higher transition probability. (*Applications in Operating Systems*)
* **Sampling Theory:** Sampling, Sampling distributions, standard error, test of hypothesis for means and proportions, confidence limits for means, student's t-distribution. Chi-square distribution as a test of goodness of fit.

### Text & Reference Books
* **Textbooks:**
  1. B.S. Grewal, *Higher Engineering Mathematics*, 43rd Edition, Khanna Publishers, 2015.
  2. Erwin Kreyszig, *Advanced Engineering Mathematics*, 9th Edition, Wiley Publications, 2013.
  3. Seymour Lipschutz, John J. Schiller, *Schaum's Outline of Introduction to Probability and Statistics*, McGraw Hill, 1998.

## 3. Chemical Technology for Computing
* **Course Code:** B25AS0105
* **Course Type:** HC
* **Credits:** 3
* **Structure (LTP):** 2-1-0 (Lecture: 2, Tutorial: 1, Practice: 0)
* **Total Classes:** 32 Lecture, 32 Tutorial/Hands-on

### Course Overview
Integrates highly pertinent chemical concepts suitable for computing engineering students, highlighting structural components of science in technology. It includes instrumental analysis methods, clean/green energy storage devices, and novel materials for flexible electronic circuits.

### Course Objectives
1. Understand electronic transitions and analytical instruments.
2. Illustrate renewable energy devices like batteries and supercapacitors.
3. Learn semiconducting materials, advanced polymeric configurations, and metal finishing.

### Course Outcomes (COs)
* **CO1:** Demonstrate the principles and uses of clean-green energy storage and conversion technologies.
* **CO2:** Interpret instrumental analytical methods and detection using sensors.
* **CO3:** Identify the significance of modern materials in electronic devices and electroplating.
* **CO4:** Analyze the importance of polymeric materials and nanomaterials.

### Course Content
* **Clean Energy Storage and Conversion Device Technologies:** Electrode System types. Batteries: Primary (dry cell), secondary (lead-acid, Li-ion battery). Assembly and fabrication of a galvanic cell. E-vehicle battery systems.
* **Supercapacitors & Photovoltaics:** Electrochemical double layer (Helmholtz model), EDLC mechanism, pseudo and hybrid capacitors. Photovoltaic cell working principles. Dye-sensitized solar cell assembly.
* **Sensors & Instrumentation:** Principles of thermal, piezoelectric, electrochemical, bio, and gas sensors. UV-Visible spectrophotometry, flame photometry, potentiometry, and conductometry.
* **Industrial & Electronic Materials:** Electroless plating of copper for Printed Circuit Boards (PCBs). Biodegradable/biocompatible polymers. Conducting polymers (polyacetylene, polyaniline - PANI) and transport mechanisms. Liquid crystals in displays. Nanomaterials size-dependent properties and quantum confinement. Smart materials: Solvatochromism, Mechanochromism, shape memory alloys.

## 4. Advanced C Programming with Generative AI
* **Course Code:** B25CI0201
* **Course Type:** HC (Integrated)
* **Credits:** 3
* **Structure (LTP):** 1-1-1 (Lecture: 1, Tutorial: 1, Practice: 1)
* **Contact Hours:** 28 Hours Theory, 42 Hours Practical
* **Assessment:** CIE 50% + SEE 50%

### Course Overview
The integration of theory and laboratory components into the Advanced C Programming with Generative AI course is a strategic necessity to evolve with modern pedagogical and industry standards. Since the foundational theory of C programming was comprehensively covered in the Introduction to C Programming (ICP) course during the first semester, this integrated model shifts the focus from passive lecturing to active, application-based mastery. Labs serve as the primary assessment engine, replacing traditional written exams with continuous, program-execution-based evaluation conducted exclusively on the Moodle platform with autograded functionality.

### Course Objectives
1. **Implement Modular and Testable Code:** Master the creation of modular, reusable code structures using both call-by-value and call-by-reference functions, adhering to the principles of clean coding and Test-Driven Development.
2. **Manage Complex Data:** Develop expertise in organizing and processing complex data by applying modular programming techniques to Arrays of Structures.
3. **Memory Management:** Gain a practical understanding of pointers and dynamic memory allocation, allowing for the creation of efficient, flexible, and memory-aware C applications.
4. **Handling Text and Processing Files:** Understand text-based data processing through user-defined string logic and file handling systems for persistent data storage.
5. **Integrate AI-Assisted Development:** Foster proficiency in using Generative AI as a collaborative tool to assist in code analysis, iterative design, and the construction of modular mini-projects through effective prompt engineering.
6. **Achieve Progressive Mastery:** Progress through a tiered learning structure (Level 1 to Level 3) designed to transition students from foundational execution to mastery-level programming through intensive practice and continuous assessment.

### Course Outcomes (COs)

| CO | Statement | Bloom's Level |
|----|-----------|:---:|
| **CO1** | Develop modular and reusable code using functions (call by value and call by reference), arrays and structures using illustrative programs and test-driven development. | Apply (L3) |
| **CO2** | Apply modular programming to manage complex data collections using Arrays of Structures through illustrative programs and test-driven development. | Analyze (L4) |
| **CO3** | Develop modular and reusable code using pointers and dynamic memory allocation using illustrative programs and test-driven development. | Apply (L3) |
| **CO4** | Develop modular and reusable code utilizing Strings by implementing user-defined logic for processing text-based data through illustrative programs and test-driven development. | Apply (L3) |
| **CO5** | Develop a modular application for file handling in C using illustrative programs and test-driven development. | Understand (L2) |
| **CO6** | Leverage generative AI to design and build a mini-project application, utilizing iterative prompt engineering and code analysis. | Create (L6) |

### Course Content (Theory)
1. **C Fundamentals:** Quick review of basic C programming concepts (arrays, structures, array of structures and functions). Function prototypes, definition, and scope (local vs. global). Parameter Passing Mechanisms: Call by value and call by reference through modular programming approach. Error handling.
2. **Array of Structures:** Passing arrays to functions as parameters, operators to access structure members, passing structures to functions as parameters, passing array of structures to functions as parameters.
3. **Pointers:** Understanding pointers and Dynamic Memory Management, using `malloc`, `calloc`, `realloc`, and `free`. Pointers and arrays, handling dangling pointers and preventing memory leaks, self-referential structures, singly linked lists as an application of pointers.
4. **Text Processing:** Reading and manipulating strings using user-defined functions and pointers.
5. **File Handling:** File handling functions — `fopen()`, `fread()`, `fgets()`, `fwrite()`, `fprintf()`, `fclose()`, `feof()`, `ferror()`. Different modes of operations (r, w, a, binary).
6. **Mini-Project using AI Tools and Prompt Engineering:** Utilizing all above concepts; leveraging Generative AI for iterative design and code analysis.

### Pedagogy — Three-Level Mastery Structure

The course is structured around **30 programs** organized across three progressive levels, all delivered in the regular classroom by faculty:

| Level | Focus | Programs | Transition |

| **Level 2 — Intermediate** | Complex applications | 10 | Continuous progression |
| **Level 3 — Mastery** | Advanced integration | 10 | Mini-project |

* **Faculty** complete 2 programs weekly in the regular classroom post week 1.
* **Slow learners** (unable to complete Level 1 in first 6 weeks) receive online scaffolding.
* **Advanced learners** attend daily online mastery training for Levels 2 and 3.

## 5. Electronics and Digital Logic
* **Course Code:** B25EE0101 (Theory) & B25EE0102 (Lab)
* **Course Type:** HC
* **Credits:** 3 (Theory) + 1 (Lab)
* **Structure (LTP):** 3-0-0 (Theory), 0-0-1 (Lab)

### Course Overview
Provides comprehensive knowledge of semiconductor configurations, digital system design, and sequential circuit logic tracking.

### Course Content
* **P-N Junction Diode:** Quantitative theory of P-N junctions, diode equation, V-I characteristics, static and dynamic resistance profiles.
* **Rectifiers & Wave Shaping:** Half-wave, full-wave, and bridge rectifiers. Diode-based clipping and clamping circuits.
* **Boolean Algebra & Gates:** Number system conversions, logical operations, Boolean theorems, simplification of logical functions, standard canonical formats (SOP, POS).
* **Gate-Level Minimization:** Karnaugh maps (2, 3, and 4 variables), don't-care conditions, universal logic gate implementations (NAND, NOR).
* **Combinational Logic Circuits:** Half/Full Adders & Subtractors, Ripple Carry Adders, Carry-Lookahead Adders, Multiplexers, Demultiplexers, Encoders, Decoders, and Magnitude Comparators.
* **Sequential Logic Layout:** Latches (SR), Flip-flops (D, T, SR, JK), Master-Slave configurations, shift registers, and counter operations.

### Lab Activities (B25EE0102)
1. Verification of Half-wave and Full-wave rectifiers.
2. Waveform clipping and clamping electronic layout.
3. Truth table verification of primitive and universal logic gates.
4. Proving De Morgan's Laws for two-variable configurations.
5. Building Adder and Subtractor circuits via discrete gates.
6. Multiplexer and Demultiplexer function logic validation.
7. Counter and Flip-flop state execution maps (JK, D, T).
