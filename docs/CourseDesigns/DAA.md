* [cite_start]**Prerequisites:** Data Structures [cite: 50]
* [cite_start]**Description:** A conceptual course focused on algorithm design techniques, time and space complexity analysis, and evaluating trade-offs between different algorithms. [cite: 15, 18]

## 4. Course Content & Syllabus
| Unit | Syllabus Topics | Weightage |
| :--- | :--- | :--- |
| **1** | [cite_start]Notion of Algorithm, Asymptotic Notations, Mathematical Analysis of Recursive/Non-recursive Algorithms. [cite: 20] | 25 Marks |
| **2** | [cite_start]Brute Force (Bubble/Selection Sort), Decrease-and-Conquer (Topological Sort), Divide-and-Conquer (Merge/Quick Sort), Transform and Conquer (Heaps). [cite: 20] | 25 Marks |
| **3** | [cite_start]Space-Time Trade-offs (Hashing), Dynamic Programming (Knapsack, Warshall’s/Floyd’s), Greedy Technique (Prim’s, Kruskal’s, Dijkstra’s). [cite: 20] | 25 Marks |
| **4** | [cite_start]Backtracking (N-Queens, Hamiltonian), Branch-and-Bound, P/NP/NP-Complete/NP-Hard problems. [cite: 20] | 25 Marks |
