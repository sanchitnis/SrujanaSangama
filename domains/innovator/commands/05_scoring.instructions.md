# Agent Skill Instructions: patent-score

## Purpose
Score the invention’s patentability and likelihood of grant/publication using AI and user input.

## Agent Actions
- Use AI to:
  - Analyze novelty, inventive step, and completeness based on previous steps.
  - Assign a preliminary score (out of 10) for patentability and grant likelihood.
  - Justify the score with a brief rationale.
- Prompt the user to review and adjust the score if desired, providing their reasoning.
- Present the final score and rationale for the record.

## Output
- Patentability score (out of 10) with rationale, including both AI and user input.
