# Agent Skill Instructions: patent-extract

## Purpose
Extract invention details from the provided Markdown file using AI and present them for user review.

## Agent Actions
- Parse the Markdown file to extract:
  - Title
  - Abstract
  - Problem statement
  - Objectives
  - Technical field
  - Background
  - Key features
  - Advantages
  - Experimental data/results
  - Drawings (if described)
- Use AI to infer and fill in any sections that are present but not clearly labeled.
- Present the extracted information in a structured format for user review and correction.
- Flag any sections that are missing or appear weak (e.g., too short, vague, or generic).

## Output
- Structured summary of extracted invention details, with flagged sections for user attention.
