# Agent Skill Instructions: patent-patentability

## Purpose
Assess the invention for Indian patentability criteria and flag missing or weak sections.

## Agent Actions
- Analyze the extracted details for:
  - Novelty (is it new compared to known prior art?)
  - Inventive step (is it non-obvious to a person skilled in the art?)
  - Industrial applicability (can it be made or used in industry?)
- Use AI to:
  - Check for completeness and clarity in each section.
  - Flag any missing, unclear, or weak sections (e.g., insufficient technical detail, lack of inventive step).
  - Suggest improvements or questions for the user to clarify weak points.
- Present a summary of the patentability assessment to the user.

## Output
- Patentability assessment report with flagged issues and suggestions for improvement.
