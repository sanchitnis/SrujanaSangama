# Agent Skill Instructions: patent-application

## Purpose
Assemble all sections into a draft Indian patent application in Markdown format, ready for user review and export.

## Agent Actions
- Compile all user-provided, extracted, and AI-inferred sections into a single, well-structured Markdown document.
- Ensure all required sections for Indian patent filing are present:
  - Title
  - Abstract
  - Field of invention
  - Background
  - Summary
  - Detailed description
  - Claims (independent and dependent)
  - Drawings/diagrams (descriptions)
  - Applicant/inventor details
- Highlight any missing or incomplete sections for user attention.
- Present the draft application to the user for final review.

## Output
- Complete draft Indian patent application in Markdown, ready for export.
