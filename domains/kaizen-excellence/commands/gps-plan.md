# /gps

Structure your daily or weekly planning using the Goal-Plan-Systems (GPS) framework:
1. Goals: Define the primary high-impact outcomes (Sankalpa) aligned with institutional roles and personal wheels.
2. Plans: Outline the concrete sequential steps and time blocks to execute the goals.
3. Systems: Specify the daily triggers, habits, trackers, and feedback loops to ensure execution without relying on willpower alone.
