# Patent Generator Plugin Workflow

## 1. Input Acquisition
- Accepts PDF, Word, or Markdown documents.
- Optionally, user specifies field/area of invention.

## 2. Information Extraction
- Extracts: title, abstract, summary, problem, objectives, technical field, background, features, advantages, experimental data, drawings.

## 3. Patentability Assessment
- Checks for novelty, inventive step, industrial applicability.
- Flags missing/weak sections for user input.

## 4. Prior Art Search
- Searches Indian/global patent databases for similar inventions.
- Summarizes closest prior art and overlaps.

## 5. Human-in-the-Loop Brainstorming
- Presents findings to user.
- Interactive Q&A to identify unique aspects, suggest pivots, refine claims.

## 6. Novelty & Patentability Scoring
- Rates invention’s patentability (out of 10) based on novelty, inventive step, clarity, relevance.

## 7. Drafting Input File for Authors
- Generates structured template for missing details: description, claims, drawings, applicant info.

## 8. Patent Application Generation
- Compiles draft Indian patent application (Form 1, 2, 3).
- Includes all required sections and guidance.

## 9. Export & Guidance
- Exports draft as DOCX/PDF.
- Offers links to Indian Patent Office resources.
